的取值范围

线性回归的目的是要使数据点尽可能地围绕在直线周围，即在这些数据点出现的地方，线性方程给出的估计值 应该充分接近观测值（真实值） 。

实现这一点的方法是对均方误差函数（mean square error, MSE）关于方程的系数求导（也就是那一堆 和 ，这里和高中经常做的对 求导有不同）。之后求出导函数等于零时的解（即高中所学的求函数的极值点），便得到了线性方程的一系列参数（斜率和截距），完成这些步骤得到完整的线性方程的过程，称为“用线性方程对数据进行拟合”。

❗但我们同时看到，在sklearn的官方文档中提到 可能取负值：

Best possible score is 1.0 and it can be negative (because the model can be arbitrarily worse).

最好的分数是 1.0，它可以是负数（因为模型可以任意变差）。

这句话并不好理解，但是参考维基百科的说法一下子就明白了：

There are cases where  can yield negative values. This can arise when the predictions that are being compared to the corresponding outcomes have not been derived from a model-fitting procedure using those data. Even if a model-fitting procedure has been used,  may still be negative, for example when linear regression is conducted without including an intercept, or when a non-linear function is used to fit the data. In cases where negative values arise, the mean of the data provides a better fit to the outcomes than do the fitted function values, according to this particular criterion.

在某些情况下，可以产生负值。当与相应结果进行比较的预测不是从使用这些数据的模型拟合过程中得出时，可能会出现这种情况。即使使用了模型拟合过程，也可能仍然是负的，例如，在进行线性回归时忽略了截距项，或当使用非线性函数来拟合数据时。根据这一特定标准，在出现负值的情况下，数据的平均值比拟合函数值更适合结果。

稍微再解释一下：

  • 如果你在A数据上训练了一个线性方程，放到B数据上去用（B和A的数据分布有明显的不同），得到的效果可能比直接用B数据训练集上的均值来预测其测试集上的样本得到的效果更糟糕；
  • 如果你的方程本来应该有截距，但是你坚持要使用没有常数项的方程来拟合这个关系，那么很可惜，你的结果可能不如用均值来乱猜（可以脑补一下这种凌乱的画面）；
  • 非线性函数的问题情况相对多一些，这里暂不展开.

我们在处理实际问题的时候应该怎么做？

  • 保证训练集和测试集来自于同一分布；
  • 允许线性方程拥有常数项。如果它不需要这一项，它会悄悄给这一项赋一个很小的值，不会给你造成额外的麻烦.
